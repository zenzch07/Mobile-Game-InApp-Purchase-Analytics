# Mobile-Game-InApp-Purchase-Analytics


##  Project Overview  
This project analyzes a simulated In-App Purchases dataset for mobile games with the objective of generating actionable business insights. By examining user engagement, recency, and revenue patterns, the analysis uncovers key drivers of monetization and player behavior. Insights are presented through interactive dashboards and reports designed to support data-driven decision-making.


##  Tools
- **Python** – data cleaning, EDA, and scripting  
- **SQL** – database querying and data extraction to quantifiable business metrics
- **Power BI** – interactive dashboards and visualizations  



- ##  Project Highlights  
- **Data Cleaning & Preparation:** Cleaned and integrated raw data into analysis-ready format.  
- **Exploratory Data Analysis (EDA):** Performed EDA to uncover patterns in engagement levels, spending patterns and main drivers of revenue.          
- **SQL Analysis:** Queried and aggregated data to compute key metrics.  
- **Visual Dashboard:** Built Power BI reports to visualize trends and insights for stakeholders.  
- **Insights:** Identified key genres by revenue, user behaviour patterns and customer segments.
